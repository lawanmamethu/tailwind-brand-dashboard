# Nexura Brand UI - Tailwind CSS Framework

![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white)
![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

A modern brand dashboard UI built with **Tailwind CSS**, featuring a completely customized theme beyond default styles. Includes responsive navigation, hero section, feature cards, pricing tables, testimonials, and footer.

---

## 🎨 Custom Brand Theme

| Element | Color | Hex Code | Usage |
|---------|-------|----------|-------|
| Primary | Coral Red | `#FF6B6B` | Buttons, borders, active states |
| Secondary | Sunset Orange | `#FF8E53` | Gradients, hover effects |
| Accent | Sun Yellow | `#FFD93D` | Icons, highlights |
| Dark Navy | Dark Navy | `#1a1a2e` | Background start |
| Midnight Blue | Midnight Blue | `#16213e` | Background middle |
| Deep Blue | Deep Blue | `#0f3460` | Background end |

### Gradient Combinations

```css
/* Primary Gradient */
background: linear-gradient(135deg, #FF6B6B, #FF8E53);

/* Text Gradient */
background: linear-gradient(135deg, #FF6B6B, #FF8E53, #FFD93D);
background-clip: text;
-webkit-background-clip: text;
color: transparent;
```

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎨 **Custom Tailwind Theme** | Extended colors, fonts, and animations beyond defaults |
| 📱 **Fully Responsive** | Mobile-first design with breakpoints at 768px and 480px |
| 🧭 **Responsive Navigation** | Desktop horizontal menu, mobile hamburger menu |
| 🏠 **Hero Section** | Animated gradient text, CTA buttons, stats counter |
| 📦 **Feature Cards** | 3 cards with icons, hover effects, and glassmorphism |
| 💰 **Pricing Tables** | 3 plans with featured "Most Popular" option |
| 💬 **Testimonials** | Client feedback with avatars and quotes |
| 📞 **Footer** | Links, social media placeholders, copyright |
| ✨ **Smooth Animations** | Float, glow, slide-up, pulse effects |
